Boggle is a brain-native web browser, reimagining internet access for people with severe motor impairments. Built using Steady State Visually Evoked Potentials (SSVEPs) and low-cost EEG headsets, it transforms the web into a space that can be browsed and controlled directly with brain signals.

Download GitHub

Breaking Barriers with Brain Technology

Boggle is developed as part of the BrainWeb project, and it contributes to ongoing research as well as practice in the field of assistive technology.

Free & Open Source
Works with Multiple EEG Headsets
Cross-Platform

Inclusive by Design

Boggle is designed for individuals with severe motor impairments, such as ALS or spinal injuries. Using Steady State Visually Evoked Potentials (SSVEPs) and accessible EEG hardware, it brings reliable, low-cost brain-computer interaction to the web, enabling access to the internet for all, without limitations.

Explore the Library
Makes use of ssvep-stimuli Generator Library

Generates SSVEP stimuli for Boggle using an in-house library developed for SSVEP brain-computer interfaces.

Efficient Interactive Element Selection

The algorithm efficiently segments the screen into regions, automatically detects interactive elements, and highlights them for easy selection by the user.

Integration with Different HTML Input Types

Supports interaction with buttons, links, checkboxes, radio buttons, text fields, numbers, dates, comboboxes, videos, audio, and more.

Works with multiple EEG headsets

Compatible with multiple EEG headsets so that users can utilise any headset they prefer.

BCI keyboard with autocomplete & numeric keyboards

Features an 18 stimuli keyboard with autocomplete and a numeric keyboard for efficient text and numeric entry.

Effective Bookmark & Tab Management

Manage bookmarks and tabs efficiently with intuitive controls for saving, opening, closing, and switching between web pages.

Highly Customisable

A settings page that offers extensive customisation options to tailor the user experience according to individual preferences.
