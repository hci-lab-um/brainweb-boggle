About Boggle

Version

Boggle is a brain-controlled web browser that redefines internet access for people with severe motor impairments. Powered by neural responses to visual stimulation (Steady-State Visually Evoked Potentials), it enables users who cannot rely on traditional assistive technologies, such as eye trackers and switch control, to navigate the web freely.

Disclaimer

This is a research tool and should be used with care. Boggle offers BCI-based web browsing based on Steady State Visually Evoked Potentials.

Custom SSVEP Stimuli Generation Library

All the stimuli used within Boggle was produced using this library. It was built from the ground up using CSS and WebGL to implement four cross-platform methods for generating SSVEP stimuli. These stimuli are produced using constant-period and square-wave approximation techniques, all suitable for BCI applications. The library enables users to customise the frequency, phase, colour, and pattern of the stimuli, with available patterns including solid, chequered, dot, and line.

Adaptive Switch Functionality

The physical switch is an optional feature, as users can operate Boggle without it. However, integrating an adaptive switch can enhance the systemâ€™s performance, particularly by improving classification accuracy. This is achieved by allowing the switch to segment the on-screen buttons into distinct groups each time it is pressed. In practice, the switch cycles through several states: all off (also known as read mode), followed by group 1, group 2, and so on, until reaching all on. For added visibility, the current cycle state is displayed in the status bar at the bottom of the window.
